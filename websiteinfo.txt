4. CANVAS (MAIN WORK AREA)
--------------------------------------------------------------------------------
Overview:
  - Infinite/pannable canvas for building system architecture diagrams
  - Built on top of a graph/flow library (likely ReactFlow based on repo context)
  - Supports node placement, edge drawing, and selection

Node Interactions:
  - Drag from sidebar to place a component node on the canvas
  - Click a node to select it → opens Props panel on the right
  - Press Enter or Space to select a node via keyboard
  - Arrow keys to move a selected node around the canvas
  - Press Delete to remove a selected node
  - Press Escape to cancel current selection/action

Edge (Connection) Interactions:
  - Draw edges between nodes to represent data flow / dependencies
  - Press Enter or Space to select an edge
  - Press Delete to remove a selected edge
  - Press Escape to cancel

Mini Map:
  - A mini-map overlay is shown on the canvas for navigation on large diagrams

Keyboard Shortcuts:
  - ⌘E (Cmd+E)         → Export diagram
  - ⌘↵ (Cmd+Enter)     → Simulate (trigger load simulation)

Empty State / Onboarding Hero Text:
  - Headline: "Build an architecture that scales"
  - Subtext: "Pick a problem, drop infrastructure components onto the canvas,
              and get scored the way an interviewer would evaluate you."
  - CTA pills:
      → "Pick a problem — 35 real interview questions"
      → "Load reference — Open a sample solution"
      → "Practice interview — Timed 6-phase mock"
  - Hint text: "Drag from the sidebar · ⌘E export · ⌘↵ simulate"
